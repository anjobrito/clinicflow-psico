# ClinicFlow Psico

Sistema web responsivo para psicólogos e clínicas de psicologia administrarem pacientes, agenda, atendimentos, evoluções clínicas, relatórios e rotinas administrativas.

## Produto

O ClinicFlow Psico será desenvolvido em Next.js com interface moderna, responsiva e fácil de usar em computador, tablet e celular.

## Módulos iniciais

- Dashboard
- Agenda
- Pacientes
- Atendimentos
- Editor de evolução
- Relatórios
- Configurações
